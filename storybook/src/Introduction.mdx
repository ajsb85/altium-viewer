# Altium Viewer Storybook

Welcome to the Storybook for the Altium Viewer Vue components.

## Components

This Storybook documents the Vue components used in the Altium Viewer application, including:

- **Core Components**: AppSidebar, AppHeader, etc.
- **UI Library**: AfsSidebar, AfsText, etc.
- **Plugins**: Search, Layers, etc.

## Testing

These components are tested using **Vitest** via the Storybook test runner.
