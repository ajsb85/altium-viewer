# Altium Viewer UI Component Library

Vue 3 components for the Altium Web Viewer.

---

## Quick Start

Browse components using the sidebar:
- **Canvas** - Interactive preview
- **Docs** - Auto-generated documentation  
- **Controls** - Adjust props in real-time

## Components

### Core UI

- **AfsBadge** - Labels and status
- **AfsLink** - Styled links
- **AfsContextMenu** - Dropdowns

### Layout

- **AfsSidebar** - Resizable sidebar
- **AppAlert** - Full-screen alerts
- **AppLoader** - Loading overlay

### Plugins

- **BoardItemsVisibility** - Layer visibility
- **SearchPanel** - Search interface
- **DownloadPanel** - File downloads
- **MeasurementDisplay** - Distance display

## Development

```bash
npm run storybook      # Start dev
npm run build-storybook # Build static
```
