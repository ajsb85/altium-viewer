The `SCHView` component is the primary viewport for rendering schematic diagrams. It combines the graphics canvas with navigation tools for zooming and switching between multiple schematic sheets.

## Features
- **Sheet Navigation**: Tabbed interface for switching between multiple schematic sheets.
- **Canvas Rendering**: High-performance rendering area for schematic vectors.
- **Zoom Controls**: Integrated toolbar for zooming in, out, and fitting the view.
- **State Feedback**: Built-in loading overlays.

## Usage

```html
<SCHView
  :sheets="[{ id: '1', name: 'Main' }, { id: '2', name: 'Power' }]"
  active-sheet="1"
  @sheetChange="handleSheetChange"
  @zoom="handleZoom"
/>
```
