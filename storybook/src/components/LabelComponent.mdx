The `Label` component provides consistent typography and styling for field labels, status text, or other short auxiliary text elements.

## Features
- **Flexible Content**: Accepts text via props or a default slot for custom markup.
- **States**: Includes a `disabled` state interaction style.
- **Typography**: Applies standard application typography settings.

## Usage

```html
<Label text="Username" />

<!-- OR with slot -->
<Label>
  Required <strong>Field</strong>
</Label>
```
