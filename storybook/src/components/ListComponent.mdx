The `List` component is a flexible utility for rendering vertical lists of items where each row can contain multiple cells of different types (text, icon, color swatch).

## Features
- **Flexible Content**: Cells can be configured as text, icons, or color swatches.
- **Selection State**: Built-in styling for selected rows.
- **Ellipsis Support**: Text truncation for long values.

## Usage

```html
<List
  :items="[
    {
      cells: [
        { type: 'icon', value: 'file-16' },
        { value: 'Document.pdf', ellipsis: true }
      ],
      isSelected: false
    }
  ]"
/>
```

## Interfaces

### ListItem

```typescript
interface ListItem {
  cells: ListCell[];
  isSelected?: boolean;
}
```

### ListCell

```typescript
interface ListCell {
  type?: 'text' | 'icon' | 'color'; // Defaults to 'text'
  value: string;                    // Content (text, icon name, color hex)
  ellipsis?: boolean;               // Enable text truncation
}
```
