The `PreviewPanel` is a container component designed to display a vertically scrollable list of document previews. It is typically used for schematic sheet navigation, allowing users to browse through multi-sheet designs visually.

## Features
- **Virtualization Support**: Efficiently handles large lists of items.
- **Lazy Loading**: Integrated image lazy loading for optimal performance.
- **Grouping**: Supports optional group headers to categorize items.
- **Selection**: Manages active item state and navigation events.

## Usage

```html
<PreviewPanel
  :items="[
    { id: '1', title: 'Top Sheet', imageSrc: 'thumb1.png' },
    { id: '2', title: 'Power', imageSrc: 'thumb2.png' }
  ]"
  selected-id="1"
  @select="selectSheet"
/>
```
