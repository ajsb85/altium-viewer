# AfsIcon Component

The `AfsIcon` component renders SVG icons from a sprite sheet. It is the core mechanism for displaying iconography throughout the application.

## Features
- **Efficient Loading**: Uses SVG sprites (`<use href="...">`) for optimal performance.
- **Scalable**: Inherits font-size from parent, allowing easy resizing.
- **Colorable**: Inherits text color (`currentColor`) for easy theming.
- **Filled Variant**: Supports an alternate "filled" state for active/selected icons.

## Usage

```html
<AfsIcon name="settings-16" />
<AfsIcon name="star-16" filled />
```

## Props

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `name` | `string` | **Required** | The ID of the icon in the sprite sheet (e.g., `'search-16'`). |
| `filled` | `boolean` | `false` | If true, applies a specific filtered/filled style (typically primary color). |

## Styling
The icon size is determined by the `font-size` of its container or the icon element itself. The color defaults to the current text color but can be overridden via CSS `color` property.

```css
.my-button {
  font-size: 24px; /* Icon will be 24px */
  color: red;      /* Icon will be red */
}
```
