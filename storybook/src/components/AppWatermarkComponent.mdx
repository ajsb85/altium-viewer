# AppWatermark Component

The `AppWatermark` component displays the "Powered by Altium 365" branding. It is typically positioned as an overlay in the viewer area and links to the relevant landing page.

## Features
- **Branding**: Official SVG graphic.
- **Positioning**: Configurable for any of the four corners.
- **Interactivity**: Hover effects and link behavior (`target="_blank"`).
- **Responsive**: Adapts opacity based on context constants.

## Usage

```html
<AppWatermark position="bottom-right" />
```

## Props

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `href` | `string` | `'https://altium.com/viewer'` | Destination URL when clicked. |
| `position` | `'top-left' \| 'top-right' \| 'bottom-left' \| 'bottom-right'` | `'bottom-right'` | Corner placement of the watermark. |
