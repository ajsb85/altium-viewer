The `LayerItem` component helps users manage layer visibility within the design. It is designed to be part of a list (e.g., in the Layers panel) and provides quick access to common visibility actions.

## Features
- **Visibility Control**: Toggle individual layer visibility or isolate a layer ("Only" mode).
- **Selection**: Integrated checkbox for batch operations.
- **Visual Identification**: Displays a color swatch matching the layer's rendering color.
- **Interaction**: Hover states reveal additional controls like the "Only" button.

## Usage

```html
<LayerItem
  name="Top Copper"
  color="#ff0000"
  :is-visible="true"
  :is-checked="false"
  @toggle="toggleLayer"
  @only="showOnlyLayer"
  @check="updateSelection"
/>
```
