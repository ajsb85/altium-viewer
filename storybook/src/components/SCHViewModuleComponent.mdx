The `SCHViewModule` is the core rendering unit for a single schematic sheet. It handles the drawing grid, zoom transformations, and displays sheet metadata overlay.

## Features
- **Grid System**: Renders a dynamic background grid that scales with zoom levels.
- **Info Overlay**: Displays current sheet name and zoom percentage.
- **Transformations**: Manages visual scaling based on zoom props.

## Usage

```html
<SCHViewModule
  sheet-name="Power.SchDoc"
  :zoom="1.5"
  :grid-size="20"
  :show-grid="true"
/>
```
