The `GridContainer` is a fundamental layout component acting as a flexbox container with column direction. It creates a full-height structural wrapper, commonly used for sidebars, panels, and application sections that require vertical stacking of child elements.

## Features
- **Flex Column Layout**: Sets `display: flex` and `flex-direction: column` for children.
- **Full Height**: Occupies 100% of the parent container's height.
- **Lightweight**: Minimal style overhead, serving primarily as a structural utility.

## Usage

```html
<GridContainer>
  <div>Header</div>
  <div style="flex: 1">Scrollable Content</div>
  <div>Footer</div>
</GridContainer>
```
