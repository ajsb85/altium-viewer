The `ObjectsPanel` provides a centralized control for managing the visibility of different PCB object types (e.g., Pads, Tracks, Vias). It supports granular control over individual object types as well as a master toggle for global visibility.

## Features
- **Object Type Management**: Lists all available PCB object types with visibility controls.
- **Bulk Control**: "All Objects" toggle for quickly showing or hiding everything.
- **Isolation Mode**: "Show Only" functionality to isolate specific object types.
- **Granular Visibility**: Individual toggles for each object type.

## Usage

```html
<ObjectsPanel
  :objects="[
    { id: 'pads', name: 'Pads', isVisible: true },
    { id: 'vias', name: 'Vias', isVisible: false }
  ]"
  :all-objects-visible="false"
  @toggleObject="handleToggle"
  @showOnly="handleShowOnly"
  @toggleAllObjects="toggleAll"
/>
```
