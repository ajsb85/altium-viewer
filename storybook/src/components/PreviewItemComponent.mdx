The `PreviewItem` component renders a single schematic sheet preview. It combines a lazy-loaded thumbnail image with a title and optional icon, used primarily in navigation lists for selecting different sheets.

## Features
- **Lazy Loading**: Integrated `LibLazyImage` handles efficient image loading.
- **Selection State**: Visual indication for active/selected items.
- **Icon Support**: Optional icon display alongside the title.
- **Responsive**: Adapts to container width while maintaining aspect ratio.

## Usage

```html
<PreviewItem
  title="[1] Power Supply"
  imageSrc="/thumbnails/sheet1.png"
  :is-selected="isActive"
  @click="handleSelect"
/>
```
