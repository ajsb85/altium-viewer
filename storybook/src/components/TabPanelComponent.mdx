The `TabPanel` component provides a standard UI for switching between different content views within a panel. It features a tab bar with support for badges and automatic content switching based on the active tab ID.

## Features
- **Tab Navigation**: Clean interface for view switching.
- **Badge Support**: Integrated badges for displaying counts or alerts on tabs.
- **Scoped Slots**: Provides the active tab ID to the default slot for dynamic content rendering.

## Usage

```html
<TabPanel
  v-model="activeTabId"
  :tabs="[
    { id: 'general', name: 'General' },
    { id: 'advanced', name: 'Advanced', badge: '!' }
  ]"
>
  <template #default="{ activeTab }">
    <div v-if="activeTab === 'general'">General settings...</div>
    <div v-if="activeTab === 'advanced'">Advanced settings...</div>
  </template>
</TabPanel>
```

## Interfaces

### Tab

```typescript
interface Tab {
  id: string;               // Unique ID for the tab
  name: string;             // Display label
  badge?: string | number;  // Optional badge content
  badgeType?: string;       // Badge style variant (e.g., 'accent')
}
```
