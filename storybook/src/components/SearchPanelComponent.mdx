The `SearchPanel` is a right-side drawer component dedicated to application-wide search functionality. It encapsulates the search header, input form, and a flexible slot for displaying results.

## Features
- **Header Actions**: Includes title, shortcut hint, and close button.
- **Search Form**: Integrated text input with submit and filter toggle buttons.
- **Flexible Results**: Uses a named slot to render arbitrary search results or states.
- **Responsive**: Configurable width to fit various layout requirements.

## Usage

```html
<SearchPanel
  title="Find Objects"
  subtitle="(Ctrl+F)"
  v-model="query"
  @search="handleSearch"
  @close="closePanel"
>
  <template #results>
    <div v-for="item in results" :key="item.id">
      {{ item.name }}
    </div>
  </template>
</SearchPanel>
```
