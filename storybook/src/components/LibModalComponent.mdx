The `LibModal` creates a side-anchored overlay panel (drawer), typically used for displaying detailed information, property editors, or navigation menus without navigating away from the main view.

## Features
- **Positioning**: Anchors to either the left or right side of the screen.
- **Resizable**: Built-in resize handle support.
- **Header Slots**: Customizable header area for titles and additional controls.
- **Layout Control**: Configurable width, offset, and height behaviors.

## Usage

```html
<LibModal
  title="Properties"
  position="right"
  :width="300"
  :visible="isOpen"
  @close="isOpen = false"
>
  <div class="panel-content">...</div>
</LibModal>
```
