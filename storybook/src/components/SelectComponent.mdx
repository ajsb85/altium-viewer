The `Select` component is a toolbar toggle button that controls the selection mode. It serves as both a mode switch (enabling selection interactions) and a status indicator for the number of currently active selections.

## Features
- **Mode Toggle**: Switches selection mode on/off via v-model.
- **Count Badge**: Automatically displays a badge when items are selected.
- **Visual Feedback**: Distinct active state styling to indicate current tool mode.

## Usage

```html
<Select
  v-model="isSelectionMode"
  :selected-items="activeItems"
/>
```
