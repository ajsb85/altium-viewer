The `ObjectItem` component represents a single PCB object type (e.g., Pads, Vias, Tracks) within the Objects visibility panel. It provides controls to toggle global visibility for that specific object type.

## Features
- **Object Identification**: Displays a specific icon and label for each PCB object type.
- **Visibility Toggle**: Eye icon indicates and toggles the visible state.
- **"Show Only" Mode**: Quick action to isolate the specific object type by hiding all others.
- **Interactive States**: Hover effects reveal secondary actions like "Only".

## Usage

```html
<ObjectItem
  name="Pads"
  icon="pads"
  :is-visible="true"
  @toggle="togglePads"
  @only="showOnlyPads"
/>
```
