The `MeasurementSection` helps organize measurement data into collapsible groups. It's often used within the larger measurement panel to segment different types of properties (e.g., Geometry vs. Properties).

## Features
- **Collapsible**: Section header toggles visibility of content.
- **Key-Value Display**: Structured layout for labeled data points.
- **Monospace Values**: Ensures numerical alignment for precision data.

## Usage

```html
<MeasurementSection
  title="Geometry"
  :items="[
    { label: 'Length', value: '10.5 mm' },
    { label: 'Angle', value: '45Â°' }
  ]"
/>
```
