The `SearchInput` component provides a robust text field for search queries. It includes built-in features for clearing the input and an optional dropdown trigger for filtering search results.

## Features
- **Integrated Actions**: Clear button appears automatically when text is present.
- **Filter Support**: Optional filter toggle button for refined searches.
- **Visual Feedback**: Focus states and styled borders align with the application theme.
- **Keyboard Events**: Native support for Enter (submit) and Escape (clear) keys.

## Usage

```html
<SearchInput
  v-model="query"
  placeholder="Search..."
  active-filter-name="Components"
  :show-filters="true"
  @submit="performSearch"
  @toggleFilters="showFilterMenu"
/>
```
