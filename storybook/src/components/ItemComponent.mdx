The `Item` component (aliased as `PreviewItem`) is designed to display content previews, such as schematic sheets or PCB layers. It combines a thumbnail image with a title, subtitle, and optional icon, making it suitable for galleries or grid-based navigation menus.

## Features
- **Visual Preview**: Prominently displays a thumbnail image (supports lazy loading via `LibLazyImage`).
- **Information Hierarchy**: Structured layout for title and optional subtitle.
- **Icon Support**: Can display an icon alongside the text for additional context (e.g., file type).
- **Interactive**: Styles for hover states to indicate interactiveness.

## Usage

```html
<Item
  title="Main Sheet"
  subtitle="Rev A"
  image-src="/path/to/preview.jpg"
  icon="sch-16"
/>
```
