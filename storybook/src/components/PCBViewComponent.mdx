The `PCBView` component is the primary viewport for 2D PCB rendering. It handles the canvas initialization, loading states, and provides basic navigation controls (zoom/pan).

## Features
- **Canvas Rendering**: Manages the HTML5 canvas lifecycle for 2D graphics.
- **State Management**: Built-in displays for loading spinners and error messages.
- **Navigation**: Integrated toolbar for Zoom In, Zoom Out, and Fit to Screen.
- **Feedback**: Overlay showing current zoom level.

## Usage

```html
<PCBView
  :loading="isLoading"
  :error="errorMessage"
  @zoom="handleZoom"
/>
```
