The `Variants` component provides a list to explore different assembly variants of a design. It allows users to switch between predefined configurations (e.g., Fitted vs Not Fitted) to see how the board changes.

## Features
- **Variant Listing**: scrollable list of available design variants.
- **Selection Support**: Highlight active variant and emit selection events.
- **Metadata Display**: Shows variant name, description, and "Default" status badges.

## Usage

```html
<Variants
  :variants="[
    { id: 'v1', name: 'Production', isDefault: true },
    { id: 'v2', name: 'Prototype', description: 'With debug headers' }
  ]"
  selected-id="v1"
  @select="handleSwitchVariant"
/>
```
